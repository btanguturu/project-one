    a

    The a is an anchor element. It defines a hyperlink and is used to link to place (like one page to another or one section to another).

    Attributes

    • href: URL part; specifics the page/website the link goes to
    • rel: specifics the realationship between the current document and the linking one
    • target: specifics where to open the link (ie a new tab)
    <a> This is a a element!</a>
